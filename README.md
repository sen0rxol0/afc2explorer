macOS desktop utility that:

- Detects a jailbroken iOS device over USB  
- Connects to `com.apple.afc2` using libimobiledevice  
- Provides full filesystem browsing  
- Supports upload, download and delete  
- Includes safety protections for system paths  

Target: advanced users.

---

## Target Environment

- macOS 10.15+
- iOS 9+ with AFC2 installed
